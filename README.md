# vuejs-starting

## 준비 과정
* nodejs 설치
	- <https://nodejs.org/>
* Atom editor 설치
	- <https://atom.io/>
* yarn 설치
	```
	npm -g install yarn
	```

## Vue-cli 사용
* Vue-cli 설치
```
npm install -g @vue/cli
or
yarn global add @vue/cli
```
* 프로젝트 생성
```
vue create new-project-name
```

* Vuetify 적용 (Optional)
```
cd new-project-name
vue add vuetify
```
